# 02-IntroWinForms

## Descripción
Introducción al desarrollo de escritorio con Windows Forms. Este proyecto utiliza el estilo de código único (sin archivos Designer) para facilitar la comprensión de la creación de controles desde C#.

## Estructura del Proyecto
Organizado en la carpeta `Views/` para coherencia con el resto del curso:

- **Views/Main/**: Ventana principal (`MainWindow`) que actúa como lanzador de ejercicios.
- **Views/HolaMundo/**: Primer formulario con botones y eventos.
- **Views/Calculadora/**: Ejemplo de lógica de eventos y diseño dinámico.
- **Views/Formulario/**: Validación de datos en WinForms.
- **Views/Layouts/**: Uso de `Anchor` y `Dock` para el diseño responsivo.

## Tecnologías
- .NET 10 (WinForms)
- C# 14

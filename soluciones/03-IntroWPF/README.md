# 03-IntroWPF

## Descripción
Primer contacto con WPF (Windows Presentation Foundation). Este proyecto introduce los conceptos básicos de diseño de interfaces con XAML y lógica en C# (code-behind).

## Estructura del Proyecto
El proyecto está organizado en la carpeta `Views/` para seguir las mejores prácticas de arquitectura:

- **Views/Main/**: Ventana principal (`MainWindow`) que sirve de lanzador para el resto de ejercicios.
- **Views/HolaMundo/**: Ejemplo básico de controles y eventos.
- **Views/Calculadora/**: Uso de Grid para crear una interfaz funcional.
- **Views/Formulario/**: Controles de entrada de datos y validación básica.
- **Views/Layouts/**: Introducción a los contenedores de WPF.

## Conceptos Aprendidos
- **XAML**: Lenguaje de marcas para UI.
- **Code-behind**: Lógica asociada al diseño.
- **Ciclo de Vida**: Eventos `Loaded`, `Closing` y `Closed`.
- **StartupUri**: Control del arranque de la aplicación en `App.xaml`.

## Tecnologías
- .NET 10 (WPF)
- C# 14

# 08-Diálogos en WPF

## Descripción
Uso de diálogos estándar del sistema y creación de diálogos personalizados modales.

## Estructura del Proyecto
- **Views/Main/**: Ventana principal con todas las demostraciones de diálogos.
- **Views/Dialogs/**: Colección de ventanas de diálogo personalizadas (Usuario, Número, Personalizado).

## Tipos de Diálogos
- **MessageBox**: Información, Advertencia, Error y Pregunta.
- **Diálogos de Archivo**: `OpenFileDialog` y `SaveFileDialog`.
- **Diálogos de Carpeta**: `FolderBrowserDialog`.
- **Personalizados**: Ventanas modales con `ShowDialog()` y `DialogResult`.

## Tecnologías
- .NET 10 (WPF)
- C# 14

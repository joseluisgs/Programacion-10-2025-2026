# 04-LayoutsComponentes

## Descripción
Profundización en los contenedores de diseño (Layouts) y componentes visuales de WPF.

## Estructura del Proyecto
Organizado por ejercicios dentro de la carpeta `Views/`:

- **Views/Main/**: Lanzador principal de demostraciones.
- **Views/Ejercicio1_Grid/**: Dominio del control Grid (Filas y Columnas).
- **Views/Ejercicio2_StackDock/**: Uso de StackPanel y DockPanel.
- **Views/Ejercicio3_WrapCanvas/**: Diseño libre y fluido.
- **Views/Ejercicio4_FormComponents/**: TextBox, PasswordBox, CheckBox, RadioButton.
- **Views/Ejercicio5_Selection/**: ComboBox y ListBox básicos.
- **Views/Ejercicio6_DataGrid/**: Visualización de datos tabulares.
- **Views/Ejercicio7_TabsExpander/**: Organización de contenido avanzado.
- **Views/Ejercicio8_Resources/**: Estilos, StaticResource y DynamicResource.

## Tecnologías
- .NET 10 (WPF)
- C# 14
